smart_toy How to set up your Telegram Bot

1. Create a Bot
  1. Open Telegram and search for @BotFather.
  2. Start a chat and send the command /newbot.
  3. Follow the instructions to name your bot and choose a username.
  4. Copy the HTTP API Token provided by BotFather.
  5. Paste it into the Bot Token field below.
2. Get Group ID
  1. Create a new Group (or Supergroup) in Telegram.
  2. Add your new bot to the group as a member/admin.
  3. Send a test message in the group (e.g. "Hello").
  4. Visit this URL in your browser:
    https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
  5. Look for "chat": { "id": -100... } in the JSON response.
  6. Copy that ID (including the minus sign) to the field below.
}

key How to get your Gemini API Key

  1. Go to Google AI Studio.
  2. Sign in with your Google Account.
  3. Click on Get API key in the left sidebar.
  4. Click Create API key (you can create it in a new project).
  5. Copy the generated key string (starts with AIza...).
  6. Paste it into the Gemini API Key field below.
